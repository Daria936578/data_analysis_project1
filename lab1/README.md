# Проект анализа данных о продуктах питания

## Описание данных
Данные представляют собой таблицу с информацией о различных продуктах питания.  
Столбцы включают:
- `product_name` (название продукта) — строковый тип данных  
- `category` (категория продукта, например, фрукты, овощи, молочные) — строковый  
- `calories` (калорийность) — числовой тип данных  
- `weight` (вес порции) — числовой  
- `protein` (белки) — числовой  
- `fat` (жиры) — числовой  

Общая информация о данных:  
- Количество строк: примерно 1000  
- Порядок данных: необязательно отсортирован по какому-либо признаку

## Библиотеки используемые для анализа
- pandas — для обработки и анализа данных  
- matplotlib — для построения графиков  
- seaborn — для красивых и информативных визуализаций

## Методы предобработки
- Проверка типов данных (`df.info()`)  
- Обнаружение пропущенных значений (`df.isnull().sum()`)  
- Замена или удаление пропусков, если есть  
- Преобразование типов данных (например, `calories` и `weight` это `float`)  
- Построение графиков для анализа распределения данных и взаимосвязей  

## Пример кода для предобработки и визуализации

```python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Загрузка данных
df = pd.read_csv('lab1/food_data.csv')

# Проверка типов данных
print(df.info())

# Обнаружение пропусков
print(df.isnull().sum())

# Замена пропущенных значений средним (если есть пропуски)
df['calories'].fillna(df['calories'].mean(), inplace=True)

# Построение гистограммы калорийности
plt.figure(figsize=(8, 4))
sns.histplot(df['calories'], bins=20)
plt.title('Распределение калорийности продуктов')
plt.xlabel('Калории')
plt.ylabel('Количество продуктов')
plt.show()

# Построение boxplot для белков
plt.figure(figsize=(8, 4))
sns.boxplot(x='category', y='protein', data=df)
plt.title('Белки по категориям продуктов')
plt.xticks(rotation=45)
plt.show()
