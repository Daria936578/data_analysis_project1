# Проект анализа данных о продуктах питания

## Описание данных
Данные содержат информацию о различных продуктах питания:
- `product_name` — название продукта (строка)
- `category` — категория (фрукты, овощи, молочные продукты, мясо и т.п.) (строка)
- `calories` — калорийность на 100 г (числовое)
- `weight` — вес порции (г) (числовое)
- `protein` — содержание белков (г) (числовое)
- `fat` — содержание жиров (г) (числовое)

Объем данных: 7 наблюдений.

## Используемые библиотеки
- pandas — обработка данных
- matplotlib — визуализация
- seaborn — статистическая визуализация

## Методы предобработки
- Проверка и преобразование типов данных
- Обнаружение и заполнение пропущенных значений
- Построение визуальных графиков для анализа распределения и взаимосвязей
